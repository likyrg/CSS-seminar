Cascading Style Sheets

Применение CSS к документам HTML основано на принципах: наследования - относится к сходным свойствам (шрифт, цвет, зум)
каскадирования - обрабатывает одновременное сопоставление более одного правила

Правила:
p#paragraph1 a:hover > * { property: value; }
.login label ~ [type="text"] { property: value; }

Класс - группа с общими признаками

  1. произвольная совокупность множеств (математика)
  2. абстрактный тип данных (програмирование)
  3. один из рангов систематики (биология)
  4. социальные общности (общество)
  5. лексико-грамматическая категория (лингвистика)
  6. одноуровненая учебная группа (педагогика)
  7. определённая совокупность стилей(CSS)

Веса селекторов

  1. Теги p 0000 0001 body div a 0000 0003
  2. Классы .main > .menu 0000 0020 a:active 0000 0011
  3. ID #head #logo a 0000 0201 a:active 0000 0011
  4. Inline style style="" 0000 1000

  5. important! переносит вес на первые четыре уровня 0100 0000
  6. Селектор атрибутов "весит" как класс p 0000 0001
  7. Селектор * абсолютно невесомый

Position / Float / Display

  • Если display равно none, то float и position должны быть проигнорированы. При этом элемент не сгенерирует контейнера
  • Если position равно absolute/fixed, то display: block, а float: none. Расположение по свойствами top/right/bottom/left
  • Если float не равен none, то display: block и элемент "всплывает"
  • Иначе применяется свойство display, в том виде, в котором оно было определено
